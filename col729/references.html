Some must-know material on modern-day performance benchmarking and optimization

Highly recommend that you know this material before starting the course.
  • Efficiency with Algorithms, Performance with Data Structures, Chandler Carruth at CppCon 2014.

  • Benchmarking C++ Code, Bryce Adelstein-Lelbach at CppCon 2015. Talks about benchmarking best practices.
    • Why you may see variance in computer performance
    • Mean-median test: easy normality test
    • Time-based benchmarking: using x86 TSC, C++11 Chrono, more ideas
    • Memory benchmarking: googleperftools/TCMalloc (MALLOCSTATS), more ideas
    • x86 hardware performance counters: Intel VTune Amplifier
    • Intel Vectorization Adviser
    • Stateful Performance Tests
