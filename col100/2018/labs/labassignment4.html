Input values using 'cin'

We can use cin to accept input ftom the standard input device such as keyboard.

Required include: <iostream>

Syntax: cin >> a;  (where a is a variable)

Example: 

		#include 

		using namespace std;

		int main()
		{
			int n;
			
			// For single input
			cin >> n;
			
			// Output the entered value
			cout << n;

			return 0;
		}
